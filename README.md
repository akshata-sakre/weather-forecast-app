Weather Forecast App – API Write-up

This project uses the OpenWeatherMap API to fetch real-time weather information for cities around the world. The API provides data such as temperature, humidity, wind speed, weather conditions, and weather icons.

The application sends HTTP requests using JavaScript Fetch API and receives data in JSON format. The retrieved data is then displayed dynamically on the webpage.

API Used:
OpenWeatherMap Current Weather API

Key Features:

* Real-time weather data retrieval
* City-based search functionality
* Dynamic weather information display
* Loading and error handling states
* Responsive user interface

Technologies Used:

* HTML5
* CSS3
* JavaScript (ES6)
* Fetch API
* OpenWeatherMap API
